完全二叉堆

  • 使用完全二叉堆去实现一个优先级队列，在逻辑结构上，完全二叉堆等效于一个完全二叉树，而在物理结构上，我们使用向量去维护

插入

图中上面是逻辑结构，下面是物理结构 - 将想插入的词条插入到向量的末尾，然后将这个词条与它的父节点比较，如果优先级大于父节点，交换位置，然后继续比较，这一过程叫做上滤

	template<typename T>
	inline Rank PQ_ComplHeap<T>::percolateUp(Rank i)
	{
		while (ParentValid(i))//只要i还有父亲就循环，否则i就是堆顶，退出循环
		{
			Rank j = Parent(i);
			if (_elem[j] > _elem[i]) break;//如果父亲优先级比i大了  就跳出循环
			swap(_elem[i], _elem[j]); i = j;//否则交换数值 更新秩 
		}
		return i;
	}

删除

图中上面是逻辑结构，下面是物理结构 - 将优先级最大的元素（大顶堆的话就是堆顶）删除，然后将最后一个节点转移到堆顶 - 就新堆顶元素与他最大的孩子比较，如果小于孩子，互换，这一过程叫做下滤

template<typename T>
	inline T PQ_ComplHeap<T>::delMax()
	{
		T maxElem = _elem[0];//备份要删除的堆顶
		_elem[0] = _elem[--_size];//用末词条代替
		percolateDown(_size, 0);//对新堆顶实施下滤
		return maxElem;
		
	}

	template<typename T>
	inline Rank PQ_ComplHeap<T>::percolateDown(Rank n, Rank i)
	{
		Rank j;
		while (i != (j = ProperParent(_elem, n, i)))//只要i不是他的两个孩子中的最大者
		{
			swap(_elem[i], _elem[j]);
			i = j;
		}
	}

建堆

  • 建堆使用Floyd算法，也就是将堆中节点按照自下而上的下滤排列，其实也就是将每个节点和它的孩子节点合并

  • 外部节点不用执行这个操作，因为没有意义，我们自下而上遍历每个内部节点

	{
		for (int i = LastInternal(n); InHeap(n, i); i--)

		}
	}

    堆排序

  • 基于选择排序，之前的选择排序需要遍历未排序的序列找到最大者，O(n^2)复杂度

  • 现在可以使用一个堆来实现未排序的序列，最大元素就是堆顶，而让新的最大元素变为堆顶就是下滤算法，，不过是O(logn)复杂度

    template<typename T>
	inline void Vector<T>::heapSort(Rank lo, Rank hi)
	{
		PQ_ComplHeap<T> H(_elem + lo, hi - lo);//以待排序区间建成一个完全二叉堆
		while (!H.empty())//直到待排序的二叉堆变空
		{
			_elem[--hi] = H.delMax();//反复的将堆顶元素与末元素兑换后下滤
		}
	}

左式堆

  • 左式堆是为了更简单的实现堆合并而出现的，因为左式堆的左倾性，也就是说，左式堆失去了结构性
  • 左式堆右侧通路节点不超过O（logn）个，所以相应的合并、删除以及插入的复杂度均为O（logn）
  • 左式堆引入了空节点路径长度，左倾性意思是：左侧节点的NPL（空节点路径长度）永远大于等于右节点的NPL

构造

    PQ_LeftHeap(T* E,int n)//应该可以改进 这里使用Floyd算法 来自下而上的下滤建堆
	{
		BinNodePosi(T) r = new BinNode<T>(E[0]);
		for (int i = 1; i < n; i++)
		{

			r->lc = new BinNode<T>(E[i]);
			r->lc->parent = r;
			r = r->lc;
		}
		//开始下滤 这时候r应该是最下面一个叶子节点
		while (!r)
		{
			if (r->data < r->lc->data)
			{
				swap(r, r->lc);
			}
			r = r->parent;

		}
	}

合并

- 怎么合并？两个堆，不是一般性的，堆a>=堆b 递归的将a的右子堆与b堆合并，然后将合并后的堆作为节点a的右孩子，然后再比较左右孩子的NPL值，如果有必要，交换，以保证左倾性

//左式堆合并
template<typename T>
static BinNodePosi(T) merge(BinNodePosi(T) a, BinNodePosi(T) b)
{
	if (!a) return b;//递归基
	if (!b) return a;
	if (a->data < b->data) swap(a, b);//假定比较器已重载 确保a比b大
	a->rc = merge(a->rc, b);//将a的右子堆，与b合并
	a->rc->parent = a;//更新父子关系
	if (!a->lc || a->lc->npl < a->rc->npl)//因为要确保是左式堆 如果左孩子npl值比右边的小
		swap(a->lc, a->rc);//交换左右子堆
	a->npl = a->rc ? a->rc->npl + 1 : 1;
	return a;//返回合并后的堆顶

}

    template<typename T>
inline T PQ_LeftHeap<T>::delMax()//利用合并的删除
{
	BinNodePosi(T) lHeap = _root->lc;//左右子堆
	BinNodePosi(T) rHeap = _root->rc;

	T e = _root->data;//备份数据
	delete _root; _size--;//删除根节点

	_root = merge(lHeap, rHeap);//合并左右子堆
	if (!_root) _root->parent = NULL;
	return e;//返回原来根节点里存放的数据
	
}
