HUGO教程

Oct 11, 2019
1 min read

1.下载HUGO和GIT

  • 下载Hugo https://github.com/spf13/hugo/releases
  • 下载Git https://gitforwindows.org/

Hugo的windows版是一个二进制文件，使用很方便，解压完之后要把解压后的exe文件所在的目录放到系统环境变量中(不然你只能在exe文件所在目录下使用hugo)

Git是个很方便的工具，学会使用很重要，我们下载完Git之后就可以用它来替代CMD使用了，与Github互连也很方便

2.使用Hugo创建一个站点文件夹

打开终端(Windows下CMD或者Power Shell还有Git都可以)，进入你想创建文件的目录，输入

hugo new site blog

site后边的参数就是你想创建的文件夹的名字，这里我们随便起了blog这个名字，hugo会在这个文件夹中自动创建关于站点的各个文件

3.安装主题

可以去https://themes.gohugo.io/网站下载一个自己喜欢的主题，可以通过Git命令下载到生成文件夹下的themes文件夹中，或者去主题的Github仓库中下载压缩包，解压后记得删除-master后缀，一般主题里都有个示例文件夹，可以将里面的东西拷出来放在根目录中

4.运行本地网站

输入

hugo server

在根目录下输入上面代码然后进入http://localhost:1313就可以访问示例网站了
